Internet Access Help

The Quick Setup supports four Internet access mode for you to choose, You can select the mode on this page.

3G/4G Only
In this mode, the Device will try 3G/4G access only. WAN access is disabled.

3G/4G Preferred
In this mode, the Device will try 3G/4G access first;
When 3G/4G access fails and WAN access is valid, or when no 3G/4G USB modem is inserted, the Device would switch to WAN access;
When the Device succeeds to connect to the 3G/4G network, the Device would stop the WAN connection and switch back to 3G/4G access immediately.

WAN Preferred
In this mode, the Device will try WAN access first;
When the WAN access fails, and 3G/4G access is valid, the Device would switch to 3G/4G access;
When the Device succeeds to connect to the WAN network, the Device would stop the 3G/4G connection and switch back to WAN access immediately.

WAN Only
In this mode, the Device will try WAN access only. 3G/4G access is disabled.
